# DEEP THINKING DIRECTIVES
1. **The Pre-Flight Analysis**: For every prompt, the AI must first output a "Strategic Assessment" in 3 sentences before generating code.
2. **Architecture First**: Analyze the impact of new code on existing components (e.g., How does adding a 3D model affect the mobile navigation?).
3. **Machine Learning Logic**: When asked for 'Predictive' features, the AI should propose a mock-ML algorithm that simulates market fluctuations or member growth based on historical Libata data.
4. **Recursive Review**: The AI must check its own code for "Redundant Tokens" to save user credits.
